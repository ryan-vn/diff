# GitHub Pages 部署摘要

## 项目名称
泰坦时光服物品数据差异对比工具 (Titan Timeline Items Compare)

## 项目内容
- `index.html` - 交互式物品数据差异查看器
- `itemsparse_3.80.0.65301_to_3.80.0.65586.zhCN.json` - 物品数据差异文件
- `README.md` - 项目说明
- `.nojekyll` - GitHub Pages 配置文件

## 功能特性
✓ 搜索和筛选功能  
✓ 按类型分类（新增、修改、删除）  
✓ 分页显示  
✓ CSV导出功能  
✓ 详细字段对比  
✓ 中文界面  

## 部署说明
1. 将所有文件上传到 GitHub 仓库
2. 在仓库设置中启用 GitHub Pages
3. 选择 main 分支和根目录
4. 访问生成的 URL 使用工具

## 访问地址
部署后可通过 `https://<your-username>.github.io/<repository-name>/` 访问

## 维护
如需更新数据，只需替换 JSON 文件并提交更改。
